!!! These verbs can be followed either by infinitive or gerund begin / continue / like / prefer / start

  1. It was necessary - had to
  2. It was possible / It was permitted  - could, was / were allowed to
  3. It wasn’t necessary - didn’t have to
  4. It wasn’t permitted / it was prohibited - couldn’t, wasn’t / weren’t allowed to
  5. It’s obligatory. It’s law or rule - must / have to
  6. It’s not necessary. You can do it if you want - don’t have to
  7. It’s forbidden to do smth - mustn’t
  8. It’s allowed or possible- - can
  9. It’s a good (bad) idea to do it / advice or recommendation - should / shouldn’t
  10. It’s not allowed - can’t
